# 学习VSCode
## 辅助线
代码长度标尺，比如在第80列的地方显示标尺，方法是在settings.json里面设置 “editor.rulers”: [80]

# 设置自动换行
文件 -> 首选项 -> 设置 -> 编辑器
找到
// 控制折行方式。可以选择： - “off” （禁用折行），
- “on” （视区折行）， - “wordWrapColumn”（在“editor.wordWrapColumn”处折行）
或 - “bounded”（在视区与“editor.wordWrapColumn”两者的较小者处折行）。
"editor.wordWrap": "off",
在右侧把off换成on.
